This is a demonstration of a scroll-activated maintenance banner. When you scroll down past 100 pixels, the banner will smoothly slide down from the top of the page.

About This Feature

The maintenance banner uses a gradient background with colors #FF8652 and #ff7a3d, creating a vibrant and eye-catching design. The banner includes a Font Awesome tools icon that pulses to draw attention.

All CSS classes and IDs are prefixed with "do-" to ensure uniqueness and avoid conflicts with your existing styles.

Key Features

• Smooth slide-in animation when scrolling down

• Close button to dismiss the banner
